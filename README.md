# Meu Bolso
O **Meu Bolso** é uma aplicação web desenvolvida para auxiliar no controle financeiro pessoal.

O sistema permite o registro de receitas e despesas, gerenciamento de metas financeiras, categorização de lançamentos e geração de relatórios detalhados com exportação para Excel/CSV.

Este projeto foi desenvolvido como parte de uma atividade, demonstrando a aplicação prática de conhecimentos da stack e manipulação de dados.

# Pré-requisitos
Antes de iniciar, você precisará ter instalado:

- PHP 8.3 ou superior
- Node.js 24 ou superior
- Composer
- PostgreSQL

# Instalação e Configuração
1. Clonar o repositório
2. Instalar dependências do Backend

    composer install

3. Instalar dependências do Frontend

    npm install
    
# Execução
Para rodar o projeto localmente, você precisará de dois terminais abertos executando os seguintes comandos:

Terminal 1 (Servidor Laravel):

    php artisan serve

Terminal 2 (Compilação Vite):

    npm run dev

A aplicação estará disponível em: http://localhost:8000
